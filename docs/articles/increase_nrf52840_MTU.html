How to increase nRF52840 feather MTU length to more that 20 bytes in Arduino?

2 things need to be changed: in central side, in file ble_gatt.h, /** @brief Default ATT MTU, in bytes. */ // FZ #define BLE_GATT_ATT_MTU_DEFAULT 203 at peripheral side: in connected_callback: connection->requestDataLengthUpdate(); connection->requestMtuExchange(200);
