Research (Space is romantic!)

My research focuses on Wireless Sensor Networks and Miniature robotic swarms for space explorations. My side projects include automatic cuicirt debug, onbody sensors, etc.

Featured Projects

AstroAnt

My designed robot is ON THE MOON!!! The AstroAnt is a miniature symbiotic robotic swarm serving on the outside surfaces of spacecraft, rovers, and landers for inspection and diagnostic tasks With modular design, each robot can carry different sensors for monitoring the performance and operations of the spacecraft, rovers, and landers.

LunarWSN

LunarWSN is a sensor network that is composed of miniature cubic sensor nodes that can be ballistically deployed to the areas-of-interest from a rover, lander, or dropped to the planetary surface by a flow-flying satellite.

HexSense

HexSense is a kind of wireless sensor node that can be ballistically deployed from a rover/lander for space explorations. After deployed, the HexSense can automatically stand up and start colelcting data of interest.

Lunar Tumbleweed

"Power-free" mobility for small wireless sensor nodes working on the Moon.

LunarDeltaT

This is a new energy harvesting approach for small wireless sensor nodes working on the Lunar surface. When small wireless sensor node working on the moon, one side of the node faces sublight and the opposite is in shadow, which will create a temperature gradient on the node's surface. This new approach harvest energy from this temperature gradient across the surface to support the sensor node's operation.

MindCube

The MindCube is an interactive device designed for studying emotions. Resembling a fidget cube toy commonly used for stress and anxiety relief, the MindCube features a compact cubic shape (3.3 cm x 3.3 cm x 3.3 cm), making it small, easy to hold, and ideal for playful interaction. The MindCube can collect signals from all the input. The data collected from the MindCube can be used to study the user's real-time emotional state.

PCBPT_1.0

The PCBPT is an automatic In Circuit Debugging tool that connects the schematic and signal output directly. With the help of the PCBPT, all you need to do is selecting signals you want to measure on the schematic and check the output from measurement instrument, like an oscilloscope.

CircuitScout

A DIY Probe Testing system that helps with automatic PCB in-circuit debugging.
